# C_Programming
This repository contains basic and intermediate level C programs.   The aim is to understand programming logic, syntax, and core concepts of the C language.

## Topics Covered
- Variables and Data Types
- Input / Output
- Conditional Statements (if, else, switch)
- Loops (for, while, do-while)
- Functions
- Arrays and Strings
- Pointers (if included)
- Structures (if included)

## How to Compile and Run
1. Open terminal / command prompt in the folder containing the .c file.
2. Compile the program:
   gcc filename.c -o Myexe
3.  Run the program:
  ./Myexe
  or on Windows:
  Myexe.exe
